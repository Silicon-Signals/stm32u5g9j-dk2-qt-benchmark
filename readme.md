## Install Required Software

Download and install Qt for MCUs and Qt Creator.
Create a kit in Qt Creator using the SDK provided by Qt.
Operating System: Windows

## Set Up and Run the Project

Clone this repository or download the ZIP file:

```bash
$ git clone https://github.com/Silicon-Signals/stm32u5g9j-dk2-qt-benchmark.git
```
   
Open Qt Creator, click “Open Project” on the Welcome screen, and select the CMakeLists.txt file inside the project folder.

Choose the appropriate kit when prompted and wait for the configuration to complete. The project will then appear in the Projects panel on the left side.

Click the Run button at the bottom left (or press Ctrl + R) to build and run the project.

Once uploaded, the UI will appear on the board display.
